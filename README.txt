Contains neo4j interface, Lucene index creation, lucene querying, and precision java classes 
in Lucene_Neo4j_java_classes directory.

Java class used to extract tweets is in Tweet_Cleaner directory

Regression commands for R is in R_Regression directory.

Link to original dataset:
https://wiki.engr.illinois.edu/display/forward/Dataset-UDI-TwitterCrawl-Aug2012

Used data subset from 07-2010 to 07-2011 with only tweets that hashtags.
